# Avatar Images Guide

## Dataset Information
- **Dataset**: `data/users/users.json`
- **Approximate images needed**: ~100 avatar portraits (1 per user)
- **Naming pattern**: `user-001.png` - `user-100.png`

## Image Specifications
- Square avatars, minimum 512x512 for responsive use
- Align numbering with each `id` (user-###)
- Used anywhere profiles display an `avatar`
